# ImageViewer

C++とWin32 APIを使用して構築された、軽量で高速なバウンディングボックス（BBox）アノテーションツール兼画像ビューアです。
機械学習（物体検出）のデータセット作成を目的としており、アノテーション情報は自動的にYOLOフォーマット（`.txt`）で保存されます。

## ✨ 主な機能

* **フォルダ単位の画像管理**: 指定したフォルダ内の画像（jpg, png, bmp）を自動で読み込み、画面下部のサムネイルリストに表示します。
* **YOLOフォーマット対応**: 画像と同じディレクトリに、自動的に `<画像名>.txt` としてYOLO形式の座標データを出力・読み込みします。
* **直感的なBBox操作**:
  * **描画**: マウスのドラッグ＆ドロップで素早くバウンディングボックスを作成。
  * **編集・移動**: 描画済みのBBoxを選択し、ドラッグで位置の移動や、四隅のハンドルを掴んでサイズ変更が可能。
  * **削除**: 選択中に `Delete` または `BackSpace` キーで即座に削除。
* **クラスラベルの割り当て**: 画面右側のパネルからクラス（Class 0 ～ 9）を選択し、BBoxに適用できます。
* **画像のズーム機能**: ＋/－ボタンによる画像の拡大・縮小表示に対応しています。

## 🛠️ 動作環境・依存関係

* **OS**: Windows 10 / 11
* **言語**: C++ (C++11以降)
* **API**: Win32 API
* **開発環境**: Visual Studio 2019 / 2022
* **外部ライブラリ**: [stb_image.h](https://github.com/nothings/stb) (画像読み込み用・シングルヘッダーライブラリ)

## 🚀 使い方

1. **画像の読み込み**
   * 画面左上の `…` ボタンをクリックし、アノテーションしたい画像が入っているフォルダを選択します。
   * 画面下部のサムネイルをクリックすると、メイン領域に画像が表示されます。
2. **アノテーションの実行**
   * 右側のリストから割り当てたいクラス（例: Class 0）を選択します。
   * 画像上でマウスをドラッグし、対象物を囲むようにバウンディングボックスを引きます。
3. **BBoxの再編集**
   * 既存の赤枠の「内部」をクリックすると水色（選択状態）になります。
   * 水色の枠内をドラッグすると**移動**、四隅の四角をドラッグすると**サイズ変更**ができます。
   * 選択状態のまま右パネルで別のクラスをクリックすると、クラスIDを変更できます。
4. **BBoxの削除**
   * BBoxを選択（水色）にした状態で、キーボードの `Delete` または `BackSpace` を押します。
